# 📚 Medicina Interna - Farreras XX edición

Aplicación de escritorio que permite acceder de forma rápida y organizada a los capítulos del libro **Medicina Interna** (Farreras Rozman, XX edición). La interfaz muestra todas las secciones en tarjetas y cada una enlaza con su respectivo archivo PDF.

![Vista previa](docs/screenshot.png) *(opcional: añade una captura)*

---

## ✨ Características

- Interfaz moderna con diseño oscuro y elegante.
- 20 secciones completas + apéndice de valores de laboratorio.
- Apertura de PDFs en la misma ventana (`target="_self"`).
- Botón flotante para volver al inicio con desplazamiento suave.
- Menú de aplicación en español (Archivo, Ayuda).
- Ventana "Acerca de" con los créditos del desarrollador.
- Empaquetado nativo para Linux (`.deb`) y versión portátil (`.zip`).

---

## 🖥️ Requisitos previos

- **Node.js** (versión 18 o superior)
- **npm** (incluido con Node.js)
